# How to apply

## `keybindings.json`

* copy to the the user's profile AppData directory
  * Windows: `[SystemDrive]/Users/[Profile]/AppData/Roaming/Code/User/keybindings.json`

## `settings.json`

* copy to the the user's profile AppData directory
  * Windows: `[SystemDrive]/Users/[Profile]/AppData/Roaming/Code/User/settings.json`
